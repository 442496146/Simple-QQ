# 简单仿QQ

基础工具  maven3+ 、jdk8、tomcat8 
开发工具  eclipse、Hbuilder 
核心框架  SpringBoot2.0、Netty 4.1 
持久层  MyBatis3.4.5 
数据库  MySQL5.6
文件服务器  nginx 1.12.0 fastdfs 5.05  
前端框架  MUI H5plus
